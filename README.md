## Getting started

### Configuration

To have this application work with your Auth0 tenant, open index.html file under /example folder and make changes to *setConfig()* method.

You will need

- Auth0 Domain name
- Client ID


### Start the app

#### Using npm

`npm install` 
`npm start`

#### Using yarn

`yarn install`
`yarn start`
